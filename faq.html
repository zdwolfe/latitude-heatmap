This tool uses location data stored on Google Latitude® to generate a heatmap of your location over time.

To be of any use, this tool requires users opt-in to Google Latitude's Location History. To then start recording data, enable location history on your mobile phone, tablet, or computer.

On an Android device, first open the Google Maps application and open Settings. Then, enable the options below:

Check "Enable location history", and "Report from this device"


Then click "Location reporting" and choose "Detect your location."

The following freely available javascript libraries are used:
  • Google Maps API Client Library for Javascript
  • Google API Client Library for Javascript
  • jQuery
  • Bootstrap
  • bootstrap-datepicker
  • heatmapjs
  • datejs
  • geometa.js
  • spinjs

Using client-side OAuth authentiation, your Google credentials are kept between you and Google. To access your personal data, an Access Token is acquired from Google using the OAuth login dialog popup. Neither this Access Token nor your Google Credentials are stored on this site or any affiliate site.

To see more about how OAuth works with Google, view their documentation.
